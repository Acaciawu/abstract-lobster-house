# 正式部署说明

当前项目已经整理成可直接部署到 Render 的 Node 应用。

## 最快上线方式

1. 注册并登录 Render
2. 创建一个新的 `Web Service`
3. 选择上传当前项目，或连接 Git 仓库
4. 构建命令填写：`npm install`
5. 启动命令填写：`npm start`
6. 健康检查路径填写：`/health`
7. 部署完成后，Render 会给你一个正式公网链接

## 当前项目包含

- `index.html`：朋友点单页
- `admin.html`：你自己查看结果的页面
- `server.js`：Node 服务与提交接口
- `render.yaml`：Render 部署配置
- `data/orders.json`：本地结果数据文件

## 需要注意

当前结果存储是写在本地文件 `data/orders.json`。

这适合快速上线测试，但正式长期使用时，建议后续再升级为数据库存储，不然在重新部署、实例重建或平台清理时，历史点单结果可能丢失。
